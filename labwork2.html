Introduction

Pandas is an open-source library for simple and intuitive data manipulation and analysis in Python. It was developed by Wes McKinney in 2008 while he was working at AQR Capital Management. At the end of 2009, it was released as open source and is now actively used in the field of Big data and data science as it offers high performance and productivity to its users.

One of the strengths of Panda is that it is based on the very popular NumPy library. It provides various data structures and operations for processing numerical data and time series. In addition to that, the data produced by Panda is often used as input for plotting functions in Matplotlib, statistical analysis in SciPy, machine learning algorithms in Scikit-learn. Data scientists use it to load, process and analyze tabular data (data stored in .csv, .tsv or .xlsx format) using SQL queries.

Datas representation using Pandas

General

There are two way to represent datas using Pandas : Series and Dataframe

Series representation

A series is a one-dimensional labeled array that can contain any type of data (integers, strings, floating point numbers, Python objects, etc.). The axis labels represent the index of the series. To put it simply, a series is nothing more than a column in an Excel sheet.

The basic method for creating a series is as follows:

Exemple

   1
ser = pd.Series(data,  index=index)

where data can be a Python dictionary, an ndarray (multi-dimensional array of NumPy) or even a scalar value.

The index passed as argument is a list of labels corresponding to each row of the series.

Dataframe representation

A Dataframe data structure that organizes data into rows and columns, making it a two-dimensional data structure. You can think of it as a spreadsheet or SQL table, or a dictionary of Series objects. This is generally the most commonly used Panda object. Like a series, a Dataframe can be constructed from many different types: a dict of 1D ndarrays, lists, dicts, or Series, a two-dimensional numpy.ndarray or another Dataframe.

The Pandas Dataframe is powerful because:

  • It can easily load data from different data sources and different data formats;
  • It allows you to easily perform statistics and answer questions about the data, such as: What is the mean, median, maximum or minimum of each column? Is there a correlation between column A and column B? What does the distribution of the data in column C look like?
  • It also facilitates data cleaning by removing missing values and filtering rows or columns by certain criteria
  • It allows you to visualize the data with the help of Matplotlib. Draw bars, lines, histograms, bubbles, etc. ;
  • It allows to store the cleaned and transformed data in a CSV, another file or a database.
  • You can define your own Python functions for certain calculation tasks and apply them to your Dataframes data.

Dataframe : Exemple

   1
df= pd.DataFrame(data,  index=index, columns=columns)

where the index represents the set of row labels and columns the set of column labels

Import and export datas

There are several function to import datas from a file, depending on the file format :

  • CSV file : pd.read_csv()
  • Excel file : pd.read_excel();
  • SQL file : pd.read_sql().

Each of these functions returns a Dataframe object. To transform a Dataframe into a Serie, just transpose the Dataframe with the transpose() function.

It's possible to export the contained datas from a Dataframe to a file

  • CSV file : pd.to_csv()
  • Excel file : pd.to_excel();
  • SQL file : pd.to_sql().

Inspect datas contained in a Dataframe

Datas contained in a Dataframe df can be inspected in several ways :

  • by displaying the first lines of the Dataframe : df.head()
  • by displaying the last lines of the Dataframe : df.tail()
  • by displaying the number of columns and rows : df.shape
  • by displaying a concise summary of the Dataframe : df.info()
  • displaying a statistical summary of numerical data: df.describe()
