La fonction copy()

The copy() function allows to copy the first level elements of a list and to store them in another list. This copy is called shallow copy and works for simple types (numbers, strings, booleans)

The deepCopy() function

The deepcopy() function of the copy module allows to make a copy of the elements of all levels of a list and to store them in another list. This copy is called shallow copy and works for types such as lists of lists, data dictionaries, records,...

Data dictionaries

Introduction

Dictionaries are more flexible arrays than lists. A list references elements by giving them a position: the list associates to each element an integer position between 0 and n-1 if n is the length of the list.

A dictionary allows to associate to an element something else than an integer position : it can be an integer, a real, a string, a tuple containing immutable objects. Generally speaking, a dictionary associates with a value what is called a key of immutable type. This key will allow to find the associated value.

The main advantage of dictionaries is the optimized search for keys. For example, we list the names of the employees of a company in a list. We then want to know if a person with a name specified in advance belongs to this list. You will then have to go through the list until you find this name or go through the whole list if it is not there. In the case of a dictionary, this name search will be much faster to write and execute.

Manipulation of a data dictionary

Most of the functions available for lists are forbidden for dictionaries, such as concatenation or the multiplication operation (*). There are no more integer indices to identify elements, the only reference is their key. The following table lists the operations on dictionaries.

x in d true if x is one of the keys of d
x not in d reciprocal of the previous line
d[i] returns the element associated with the key i
len(d) number of elements of d
min(d) returns the smaller key of d
max(d) returns the larger key of d
del d [i] deletes the element associated with the key i
list (d) returns a list containing all the keys of the dictionary d
dict (x) converts x into a dictionary if possible, d is then equal to dict( d.items () )
d.copy () Returns a copy of d
d.items () Returns an iterator over all pairs (key, value) included in the dictionary.
d.keys () Returns an iterator on all the keys of the dictionary d
d.values () Returns an iterator on all the values of the dictionary d
d.get (k[,x]) Returns d[k], if the key k is missing, then the value None is returned unless the optional parameter x is specified, in which case this value will be returned.
d.clear () Deletes all the items in the dictionary.
d.update(d2) The dictionary d receives the content of d2.
d.setdefault(k[,x]) Sets d[k] if key k exists, otherwise assigns x to d[k]
