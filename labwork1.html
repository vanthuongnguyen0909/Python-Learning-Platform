Introduction to the NumPy and Pyplot modules

Introduction

The NumPy module allows you to perform numerical calculations with Python. It introduces an easier management of arrays of numbers and provides efficient support for large multidimensional arrays, and high-level mathematical routines (special functions, linear algebra, statistics, etc.).

   1
>>> import numpy as np # Import of the numpy library with the nickname np

Creating a NumPy array

Arrays with NumPy

A NumPy array (usually called array) is a homogeneous multidimensional array: all elements must have the same type, usually numeric. The different dimensions are called axes, while the number of dimensions - 0 for a scalar, 1 for a vector, 2 for a matrix, etc. - is called the rank.

Warning: do not confuse the rank of an array with the rank of a matrix!

Exemple

Creating a numpy array of numbers

A table is created with the instruction np.array().

All the elements are delimited by square brackets. Each element is separated by a comma.

   1
>>> a = np.array([1,2,3,4,5])

It is also possible to create arrays of arrays representing multi-dimensional arrays:

   1
>>> a = np.array([[1, 2, 3], [4, 5, 6]])

Creating a numpy array with the linspace function

It is possible to create a table with the function linspace(start, end, n).

This function returns a NumPy array starting with an element of value start, ending with the element of value end with n elements regularly spaced.

Exemple

   1
>>> tab = np.linspace(1., 4., 6)

   1
array([ 1. ,  1.6,  2.2,  2.8,  3.4,  4. ])

Creating an numpy array with the arange function

The function arange(start, end, step) creates a sequence of numbers starting at the start value and ending at the end value (excluded) separated by a step value.

Exemple

   1
>>> np.arange(10, 30, 5) # From 10 to 30 (excluded) by steps of 5, default integer type

   2
array([10, 15, 20, 25])

   3
>>> tab = np.arange(0.5, 2.1, 0.3) 

   4
>>> tab 

   5
array([ 0.5,  0.8,  1.1,  1.4,  1.7,  2. ])

Exercice : Loading a dataset

                    
Context

                    
The file population.csv contains, for a series of 66 individuals with for each individual, the values of sex, height and weight.

                    
The format of this file, called CSV (Comma Separated Values) format, is the standard format for exchanging statistical data between software.

                    
A CSV file can be read and loaded almost directly into an array object. You can then perform all the calculations you need to analyze this data via Python.

                    
In this format, the first line (often) gives the names of the characters studied and the following lines each describe an individual by listing the values of the characters of this individual and by separating them by commas.

                    
Loading a file into a NumPy array is done using the :

                    
  1
>>> tab = genfromtxt(open(file_name,"rb"),delimiter=",",names=True,dtype=None)

                    
avec :

                    
    
                        
  • tab, corresponds to a NumPy array;
    • 
                        
  • file_name, corresponds to the name of the file ;
    • 
                        
  • delimiter, the delimiting character between each column;
    • 
                        
  • names, a boolean indicating whether or not to keep the column names.
    •
